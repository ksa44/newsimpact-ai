# 🚀 NewsImpact AI

Predict how crypto markets react to news — before it happens.

Simulates a real multi-agent AI system that analyzes news, detects patterns, and generates market predictions.

---

## 💡 Overview

Crypto markets move fast — often driven by news, tweets, and announcements.

NewsImpact AI helps answer a key question:

**"Will this news cause a pump, dump, or no impact?"**

---

## ⚙️ How It Works

The system uses a multi-agent architecture inspired by OpenClaw.

Each agent performs a specific role:

- **Input Agent** → Receives news or tweet  
- **Sentiment Agent** → Detects positive / negative tone  
- **Event Agent** → Classifies type (hack, partnership, regulation, etc.)  
- **History Agent** → Matches similar past events  
- **Analysis Agent** → Combines all signals  
- **Decision Agent** → Outputs prediction  
- **Explanation Agent** → Generates reasoning  

---

## 🧠 Features

- 📈 Predict market reaction (Pump / Dump / Neutral)  
- 🎯 Confidence score  
- 📰 News & tweet analysis  
- 🧩 Event classification  
- 📊 Historical pattern simulation  
- 💬 AI-generated explanations
